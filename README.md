# Description
  This repository contains the files of a 32-bit RISC - V Core. 
# Features:
<ul>
  <li> 32-bit Instruction Format </li>
  <li> Implemented 27 Instructions of the RV32I Core </li>
  <li> 5 Stage Pipeline Design </li>
  <li> Hazard Handling </li>
</ul>
